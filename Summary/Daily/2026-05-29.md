# motivation

- 完成 phase-aligned perf collection。
  - `scripts/run_policy_once.sh` 新增 `--collect-perf`、`--perf-events`、`--perf-interval-ms`。
  - 开启后在 `run_phases.sh` 前启动 `perf stat -I <interval> -a -x,`，raw 输出写入 `run_dir/perf_interval.csv`。
  - `meta.txt` 记录 `perf command` 和 `perf_start_ts_ns`，便于后处理按 monotonic timestamp 对齐 phase。
  - `scripts/run_policy_matrix.sh` 支持 `--collect-perf` / `--collect-phase-perf` 并转发到单次 run。
  - `scripts/run_root_privileged_pipeline.sh` 新增 `--collect-phase-perf`，默认关闭，避免普通主路径被 perf 权限影响。

- 新增 phase-aligned perf feature 聚合。
  - 新增 `analysis/collect_phase_aligned_features.py`。
  - 读取 `phases.csv`、`perf_interval.csv` 和 `meta.txt` 中的 `perf_start_ts_ns`。
  - 将 perf interval elapsed time 映射到 LC-Serve phase。
  - 输出 `feature_by_phase.csv`，包含 `ipc`、`cache_miss_rate`、`branch_miss_rate`、`context_switches_per_sec`、`migrations_per_sec`、`runtime_sec`。
  - `analysis/collect_matrix_summary.py` 在 run 目录存在 `feature_by_phase.csv` 时自动合并这些 feature 字段。

- 验证 perf 权限和 phase-aligned perf 路径。
  - 初次测试时 `perf_event_paranoid=4` 导致 `perf stat -a` 被拒绝。
  - 调整 perf 权限后，短 run 成功生成非空 `perf_interval.csv` 和 `feature_by_phase.csv`。
  - 短 matrix run 验证 `run_policy_matrix.sh --collect-phase-perf` 能转发并生成 feature。
  - 用 `collect_matrix_summary.py` 验证 feature 字段能进入 `matrix_summary.csv`。

- 新增 FIFO side-effect experiment，用于反驳“直接使用 fg_fifo 即可”。
  - 新增 `src/background/fixed_batch_job.cpp`，执行固定规模 batch work，而不是按 duration 运行。
  - 支持 `compute` / `memory` 两种模式，支持 `--work-units`、`--threads`、`--memory-mb`、`--ready-file`、`--start-file`。
  - 输出 `fixed_batch_job.csv`，包含 `job_start_ts_ns`、`job_end_ts_ns`、`completion_time_ms`、线程数、模式、work units 和 memory size。
  - `CMakeLists.txt` 增加 `fixed_batch_job` 构建目标。

- 新增 FIFO side-effect 单次 runner。
  - 新增 `scripts/run_fifo_side_effect.sh`。
  - 同时启动 target `lc_server`、side `lc_server`、`fixed_batch_job`、target client 和 side client。
  - target server 使用 policy foreground wrapper；side server 始终 default；batch job 使用 policy background wrapper。
  - 输出 `target_client.csv`、`side_client.csv`、`target_server.csv`、`side_server.csv`、`fixed_batch_job.csv`、`policy_meta.csv`、`summary.csv`、`summary.txt` 和 `meta.txt`。
  - 支持 root/sudo 运行后的 `results/` 和 `build/` ownership 修复。

- 新增 FIFO side-effect matrix、summary 和图。
  - 新增 `scripts/run_fifo_side_effect_matrix.sh`，支持多 policy、多 repetitions，默认 policy 为 `default,fg_fifo,bg_idle,affinity`。
  - 新增 `analysis/summarize_fifo_side_effect.py`，输出 target/side e2e p99、target/side schedule lag p99、batch completion time 和请求数。
  - 新增 `analysis/plot_fifo_side_effect.py`，用 Python 标准库生成 SVG 图：
